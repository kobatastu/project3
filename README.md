# Project3

## 実行方法

git cloneした後、フロントエンド、バックエンドでそれぞれを実行する

### フロントエンド

1. pcにxcodeをインストール、携帯端末にexpoアプリをインストールする
2. 以下のコマンドを実行

```
$ cd expo
$ npm install
$ npm run start
```

3. pcのウェブ画面にQRコードが表示されるので、iOSの場合カメラアプリを起動、Androidの場合はexpoアプリでQRコードを読み込む

### バックエンド

以下のコマンドを実行

```
$ cd server
$ npm install
$ npm run start
```
